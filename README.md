# 🍳 SmartChef App

> Aplikasi web cerdas yang membantu kamu menemukan resep berdasarkan bahan yang tersedia di dapur.

---

## 📖 Tentang Proyek

**SmartChef** adalah aplikasi web full-stack yang memungkinkan pengguna memasukkan bahan-bahan yang ada di dapur dan mendapatkan rekomendasi resep yang relevan secara cerdas. Proyek ini terdiri dari dua bagian utama: frontend berbasis HTML/CSS/JavaScript dan backend berbasis Python.

---

## 🗂️ Struktur Proyek

```
smartchef-app/
├── smartchef-frontend/     # Antarmuka pengguna (HTML, CSS, JS)
│   ├── index.html
│   ├── style.css
│   └── script.js
│
├── smartchef-backend/      # Server & logika aplikasi (Python)
│   ├── app.py
│   └── requirements.txt
│
└── README.md
```

---

| Bagian | Teknologi |

## 🚀 Cara Menjalankan

### Prasyarat

- Python 3.8+
- Browser modern (Chrome, Firefox, Edge, dll.)

---

### 1. Clone Repository

```bash
git clone https://github.com/afif-jarvas/smartchef-app.git
cd smartchef-app
```

---

### 2. Jalankan Backend

```bash
cd smartchef-backend
pip install -r requirements.txt
python app.py
```

Backend akan berjalan di `http://localhost:5000` (atau port yang dikonfigurasi).

---

### 3. Jalankan Frontend

Buka file `smartchef-frontend/index.html` langsung di browser, atau gunakan ekstensi **Live Server** di VS Code untuk pengalaman terbaik.
